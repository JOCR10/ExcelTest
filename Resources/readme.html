Interview Question - Veritas/RRD Nov 4, 2020

Task

Along with this document please find a sample data file. This file is a standard CSV file with the first row as headers. This file is a set of consumer data along with a set of flags. Each row of this file consists of a set of flags and a set of addresses to which these flags apply. The flags are used for various processing purposes down the processing pipeline. The addresses are given in a so-called "wide" format. This format consists of multiple Names, Addresses, Cities, States, and Zip codes presented on the same line. Each set of data is presented with a numeral at the end (Name1, Address1, City1, etc.). Values ending with the same numeral belong as a part of the same set (Name2, Address2, City2, State2, Zip2 all go together, for example). There may be one, two, three, or four sets of these on a given line. There may be missing sets on the line in any position but the first. There will always be a set of data in the first position.
